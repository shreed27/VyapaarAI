# Transaction Stuck in Pending State

## What happened
Your payment shows as "pending" and neither you nor the recipient can confirm the money moved. This is common during high-traffic periods.

## Why it happens
- Bank servers processing a backlog of transactions
- Network interruption mid-transaction
- UPI switch congestion

## How to check status
1. Paytm app → Transaction History → find the transaction → check status
2. Check bank account balance — did it change?
3. Check UPI reference number (UTR) in transaction details

## Resolution steps
If balance was deducted but payment shows pending:
1. Wait 2 hours — most pending transactions resolve automatically
2. If still pending after 2 hours: Paytm app → Help → Dispute a transaction
3. Provide the UTR number from transaction details
4. Expected resolution: 2-5 business days

If balance was NOT deducted:
1. The transaction likely failed — retry the payment
2. No action needed for the pending transaction
