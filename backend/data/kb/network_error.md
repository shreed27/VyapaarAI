# Network Error During Payment

## Error codes: NETWORK_ERROR, U90, PSP_UNREACHABLE

## What happened
The transaction could not be completed because the internet connection or payment service connection was interrupted during processing.

## Why it happens
- Weak mobile data signal
- Wi-Fi handoff or unstable public network
- PSP gateway temporary connectivity issue
- Background app restrictions interrupted the payment session

## Resolution steps
1. Check whether your internet is stable by opening another app or website
2. If the payment status is unknown: first check bank balance and transaction history
3. Retry only after confirming the earlier attempt did not succeed
4. Switch between mobile data and Wi-Fi if one network is unstable

## Prevention
Avoid completing UPI payments on unstable public Wi-Fi when mobile data is available
