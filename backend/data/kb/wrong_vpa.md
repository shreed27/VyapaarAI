# Invalid VPA / UPI ID Not Found

## Error codes: U69, INVALID_VPA, PAYEE_NOT_FOUND

## What happened
The UPI ID (VPA) you entered does not exist or has been deactivated. No money was deducted.

## Why it happens
- Typo in the UPI ID (e.g., name@okaxis vs name@okhdfcbank)
- Recipient has deleted or deactivated their UPI ID
- Wrong bank suffix used (common: @paytm, @okaxis, @okhdfcbank, @okicici, @oksbi, @ybl, @ibl)

## Resolution steps
1. Double-check the UPI ID spelling with the recipient
2. Ask recipient to share their QR code instead of typing the ID manually
3. Verify the bank suffix is correct
4. If sending to a phone number, confirm the number is linked to a UPI account

## No money lost
This error always means the transaction was rejected before reaching the bank — no deduction occurred.
